git init - initialize repo
git status - what is the status of all of the files in this repo
git add <file> - tell git to start tracking the file
git commit -a - tell git to save a new snapshot of the tracked files (-a = all files)
git diff <file> - reveal the differences between current directory and repo
git checkout main <commit_id> <file> - revert to repo version 
git log -n <number of commits> - shows a log of all commits on these repo
